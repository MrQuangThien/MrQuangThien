## 👨‍💻 About Me

MrQuangThien là một Full-stack Web Developer với định hướng phát triển chuyên sâu về backend, thiết kế hệ thống và tối ưu hiệu năng. Anh có kinh nghiệm xây dựng các ứng dụng web hoàn chỉnh sử dụng ASP.NET Core, Django và SQL Server, kết hợp với frontend bằng HTML, CSS và JavaScript để tạo ra các sản phẩm hiện đại, hiệu quả. Hiện tại, anh đang tập trung nghiên cứu kiến trúc hệ thống nâng cao như microservices và scalable architecture nhằm phục vụ các hệ thống lớn. Bên cạnh đó, anh cũng quan tâm đến AI/Machine Learning và tích cực tham gia các dự án mã nguồn mở để nâng cao kỹ năng và đóng góp cho cộng đồng.
